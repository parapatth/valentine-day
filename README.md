# ✨ Will You Be My Valentine? ✨

A cute, interactive Valentine's card that won't take no for an answer.

## Features

- **Escape-proof "No" button** - Try clicking it. I dare you.
- **Judgmental cat memes** - Keep trying to say no and find out what happens.
- **Personalization** - Add your crush's name via URL params
- **8 adorable cats** - Choose your fighter
- **Mobile friendly** - Works on touch devices (the No button escapes there too)
- **Create page** - Let others make their own Valentine cards

## Usage

**Basic:**
```
https://yoursite.vercel.app/
```

**With name:**
```
https://yoursite.vercel.app/?name=Sarah
```

**With name and specific cat:**
```
https://yoursite.vercel.app/?name=Sarah&cat=5
```

**Create your own:**
```
https://yoursite.vercel.app/create.html
```

## Deploy

Drop it on Vercel, Netlify, or any static host. It's just HTML.

```bash
vercel
```

## What happens when you click "No"?

The button runs away. If you keep trying, you'll meet some very disappointed cats.

## What happens when you click "Yes"?

Happiness. Hearts. A very excited cat.
